## User Flow (4 Stage)

### Stage 1: 입력 및 의도 분석 (태블릿)
1. 시작 (태블릿 실행)
2. 상품검색 (음성/텍스트 입력)
3. 의도분석 (상품위치 확인)
   - No → 위치안내 불가 안내 (질문 예시 / 직원 콜)
4. 키워드 추출

### Stage 2: 검색 (태블릿)
5. 검색 실행
6. 검색 결과 판별
7. 결과 화면 출력 (예시 3개 / 결과 없음)

### Stage 3: 결과 안내 (태블릿)
8. 상품 선택
9. 매대 번호 및 QR 코드 안내
13. 태블릿 종료 (자동/다음 터치)

### Stage 4: 모바일 연동 (모바일)
10. 모바일 카메라 또는 QR 리더 실행
11. QR 스캔
12-1. 지도 보며 이동
12-2. AR 네비 보며 이동
14. 모바일 안내 종료
