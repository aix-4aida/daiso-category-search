# 어디다있소 UX/UI 가이드라인 (v3.0)

## 1. 브랜드 아이덴티티

### 로고
- 서비스명: **어디다있소**
- 로고 렌더링: `어디` (검정) + `다` (빨강) + `있` (빨강) + `소` (빨강)
- "다있소" 부분이 모두 빨간색 → "다이소" 브랜드 강조
- 좌측에 다이소 로고 아이콘 (빨간 사각형) 배치 가능

### 컬러 팔레트
| 토큰 | 색상 | 용도 |
|------|------|------|
| `--color-primary` | `#E50000` | 브랜드 빨강, CTA 버튼, 로고 강조 |
| `--color-primary-light` | `#FFEBEB` | 카드 배경, 선택 하이라이트 |
| `--color-bg` | `#FFFFFF` | 페이지 배경 |
| `--color-bg-light` | `#F5F5F5` | 섹션 구분 배경 |
| `--color-text` | `#333333` | 본문 텍스트 |
| `--color-text-light` | `#888888` | 보조 텍스트, 가격 |
| `--color-border` | `#E0E0E0` | 카드/섹션 테두리 |
| `--color-map-path` | `#2962FF` | 지도 이동경로 (파란 점선) |
| `--color-map-pin` | `#E50000` | 지도 위치 마커 |

### 타이포그래피
- 기본 폰트: `Pretendard`, fallback: `'Noto Sans KR', sans-serif`
- 제목 (h1): 28px, Bold
- 소제목 (h2): 20px, SemiBold
- 본문: 16px, Regular
- 보조: 14px, Regular, color-text-light
- 가격: 16px, Bold, color-primary

## 2. 화면 구성 (6개 핵심 화면)

### 2.1 키오스크 홈 (태블릿)
**구성 요소 (위→아래):**
1. **헤더**: 좌측 다이소 로고 + "어디다있소", 우측 KOR/ENG 토글 + 도움말 아이콘
2. **추천 상품 캐러셀**: 카드형 상품 슬라이더 ("오늘의 추천 상품", 가격 태그)
3. **음성 검색 버튼**: 중앙 큰 빨간 원형 마이크 아이콘 + 파동 애니메이션
4. **검색 안내 텍스트**: "상품 이름을 말씀해주세요"
5. **하단 탭 바**: 카테고리 지도 | 상품 위치 찾기

**인터랙션:**
- 마이크 버튼 탭 → 음성 녹음 시작 (파동 애니메이션)
- 캐러셀 좌우 스와이프 → 추천 상품 탐색
- 하단 탭 → 화면 전환

### 2.2 로딩 화면 (태블릿)
**구성 요소:**
1. **원형 프로그레스 바**: 빨간색, 중앙에 퍼센티지 (85%), dashed 스타일
2. **마스코트 캐릭터**: 빨간 쇼핑백 캐릭터 (프로그레스 바 좌측 하단에 걸침)
3. **텍스트**: "상품을 찾는 중..." (Bold)
4. **검색어 표시**: 말풍선 형태 ("규조토 욕실 매트")
5. **하단 탭**: 홈 | 장바구니

### 2.3 검색 결과 화면 (태블릿)
**구성 요소 (좌우 분할 레이아웃):**
- **좌측 패널 (30%):**
  1. "검색 결과 3개" 타이틀
  2. 상품 카드 리스트 (선택된 카드 = 빨간 좌측 보더)
     - 상품 이미지 + 상품명 + 위치코드 (📍 B1-B01) + 가격
  3. QR 코드 영역
  4. 카카오 보내기 버튼 (노란색)
- **우측 패널 (70%):**
  1. B1 / B2 매장 평면도
  2. 파란 점선 이동 경로 (현재위치 → 상품위치)
  3. 빨간 핀 마커 (상품 위치)
  4. 파란 원 마커 (현재 위치)
- **하단 탭**: 검색 | 매장지도 | My

### 2.4 카테고리 지도 화면 (태블릿)
**구성 요소:**
- **좌/중앙**: B1 Floor + B2 Floor 매장 평면도 (나란히)
- **우측 사이드바**: 카테고리 필터 버튼 목록
  - 전체, 뷰티, 주방, 욕실, 문구, 수납정리, 식품, 인테리어, 애견
  - 각 버튼: 아이콘 + 텍스트, 선택 시 빨간 배경
- **하단 탭**: 카테고리 지도 | 상품 검색 | 장바구니

### 2.5 모바일 지도 화면 (모바일)
**구성 요소:**
1. **헤더**: ← 뒤로가기 + "어디다있소"
2. **상품 정보 카드**: 이미지 + 상품명 + 위치코드 + "약 30초 소요"
3. **매장 평면도**: B1 지도 + 경로 점선 + 현재위치 마커
4. **하단 버튼**: 지도 보기 (빨간 배경) | AR 네비 (흰 배경, 빨간 테두리)

### 2.6 AR 네비게이션 화면 (모바일)
**구성 요소:**
1. **헤더**: "어디다있소" + WiFi 아이콘 + 메뉴
2. **카메라 뷰**: 전체화면 카메라 피드 (시뮬레이션)
3. **AR 오버레이**:
   - 3D 빨간 화살표 (방향 지시)
   - 거리 표시 ("12m 앞으로")
   - 위치 마커 핀 ("B1-B01")
4. **하단 상품 카드**: 상품 이미지 + 이름 + 거리 + 소요시간
5. **토글 버튼**: 지도 보기 | AR 네비

## 3. User Flow (4 Stage)

### Stage 1: 입력 및 의도 분석 (태블릿)
1. 시작 (태블릿 실행)
2. 상품검색 (음성/텍스트 입력)
3. 의도분석 (상품위치 확인)
   - No → 위치안내 불가 안내 (질문 예시 / 직원 콜)
4. 키워드 추출

### Stage 2: 검색 (태블릿)
5. 검색 실행
6. 검색 결과 판별
7. 결과 화면 출력 (예시 3개 / 결과 없음)

### Stage 3: 결과 안내 (태블릿)
8. 상품 선택
9. 매대 번호 및 QR 코드 안내
13. 태블릿 종료 (자동/다음 터치)

### Stage 4: 모바일 연동 (모바일)
10. 모바일 카메라 또는 QR 리더 실행
11. QR 스캔
12-1. 지도 보며 이동
12-2. AR 네비 보며 이동
14. 모바일 안내 종료

## 4. 컴포넌트 설계 규칙

### 버튼
| 유형 | 스타일 |
|------|--------|
| Primary | 빨간 배경 (#E50000), 흰 텍스트, border-radius: 25px |
| Secondary | 흰 배경, 빨간 테두리 1px, 빨간 텍스트, border-radius: 25px |
| Tab (활성) | 빨간 텍스트, 하단 빨간 인디케이터 |
| Tab (비활성) | 회색 텍스트 |

### 카드
- 배경: 흰색
- 테두리: 1px solid #E0E0E0
- border-radius: 12px
- 그림자: 0 2px 8px rgba(0,0,0,0.08)
- 선택 상태: 좌측 4px 빨간 보더

### 지도
- 배경: 흰색 / 연회색
- 매대 블록: 연한 배경 + 텍스트 라벨
- 이동 경로: 파란 점선 (stroke-dasharray)
- 현재 위치: 파란 원 (파동 효과)
- 상품 위치: 빨간 핀 마커

### 마이크 버튼
- 크기: 80px 원형
- 배경: 그라디언트 빨강
- 아이콘: 흰색 마이크
- 대기 상태: 그림자만
- 녹음 중: 바깥 파동 애니메이션 (빨강 반투명 원 확산)
